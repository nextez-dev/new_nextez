# Capabilities > Data & Platform

## 페이지 헤딩

**데이터를 모으고, 정리하고, 보여주는 모든 단계를 자체 플랫폼으로 다룹니다.**

## 본문 (역량 정의)

㈜넥스트이지는 데이터 수집·정제·표준화·저장·결합·시각화·개방까지 전 단계를 직접 다루는 자체 플랫폼을 보유하고 있습니다. 우리가 운영 중인 제주DA·JADX·BigData 통합 모니터링·CMS는 단순한 도구가 아니라, 제주 공공기관 71여 곳과 전국 다수의 발주처에서 실제로 가동 중인 검증된 시스템입니다.

## 4대 데이터·플랫폼 영역

- **제주DA · JADX 플랫폼**: 농업 도메인 통합 운영 플랫폼
  - 1단계 34종 데이터 허브 구축 → 2단계 신규 10종 확충 (기상청 ASOS/AWS, 태풍, 보조금, 도매시장 경락가, 서리 등)
  - 외부·IoT·내부 조사 시스템 통합 수집
  - 데이터 결합 모듈 (매칭 키·병합 조건 자동 설정)
  - 품질 검증 룰셋 (필수·정규식·범위 자동 검증)
  - 정책사업 자동 연계 (액비·농민수당·여성농업인 행복이용권·농어촌기금)
  - 공공데이터 API 개방 체계 (10종)
- **빅데이터 통합 모니터링 시스템 (C-2023-055263)**
  - 분산된 데이터의 수집·정합·품질관리·통계·시각화·총정 관리

### 02. 통합 플랫폼 & CMS
- **나이스 콘텐츠관리시스템 v1.1 (CMS, GS 1등급, 17-0563)**

  - 표준 프레임 설계 기반, 콘텐츠 발행·관리 통합
  - 유동적 콘텐츠 유형 대응, 권한 관리, 다채널 응답형
- **통합 플랫폼 웹·앱 고도화**
  - 영농일지 (작물·사용자 대상 확대, 큰글씨·홈화면·메뉴구조 개선)
  - AI 검색 (임베딩·멀티턴·통합)
  - IoT 장비 유지관리 시스템 (모바일 + 모니터링)
  - 기술지도 이력관리 + 모집단 조회

### 03. GIS·시각화 (Visualization)

### 04. 데이터 거버넌스
- 신규 데이터 수집 모듈 (표준 스키마 설계)

- 공공데이터 개방 (API 기반, 실시간 반영)

## 강조 메시지 박스 후보

### Box A. "도구를 만들었으면 운영도 합니다"
> 우리는 CMS를 만든 회사일 뿐 아니라, 그 CMS로 71여 개 공공기관 사이트를 운영 지원하는 회사입니다.
> 만든 도구로 직접 살아보지 않은 회사는 곧 한계가 보입니다.

### Box B. "AI는 좋은 데이터 위에서만 동작합니다"
> 우리가 AI 솔루션에 강한 이유는, 그 아래 데이터 허브를 직접 다루기 때문입니다.
> 데이터 결합·품질관리·자동 연계 없이는 AI 모델도 신뢰할 수 없습니다.
