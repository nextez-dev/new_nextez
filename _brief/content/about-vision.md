# About > Vision & Mission

## 페이지 헤딩

**산업 현장의 데이터로 AI를 만들고, 그 AI로 산업을 다시 만듭니다.**

## Vision (한 줄)

> **"산업을 알아야 진짜 AI를 만들 수 있고, 진짜 AI여야 산업을 바꿀 수 있다."**

## Mission

㈜넥스트이지는 25년간 관광·농업·에너지 현장을 직접 운영하며 도메인 데이터를 축적해왔습니다.
이 데이터로 컴퓨터비전·LLM·예측 모델을 만들고, 다시 공공기관과 기업의 디지털 전환(DX·AX) 전략으로 환원합니다.

## 3대 원칙 (Mission Statement)

### 01. 도메인 운영 — Operate the domain.
> 우리는 관광 27년차 여행사(jeju114), 17개 골프장 통합 플랫폼(섬프로), 라이브커머스 OTA(탐라이브)를 직접 운영합니다.
> 농업 분야에서는 제주DA·JADX 플랫폼을 농업기술원과 함께 운영하며 34종 이상의 데이터 허브를 관리합니다.
> 현장을 운영하지 않으면 보이지 않는 문제가 있습니다.

### 02. AI 솔루션 제작 — Build domain AI.
> 농업 병해충 12종 식별 모델(YOLOv8/YOLO26x), 감귤 3D 측정 파이프라인(2시간→30분), LLM 기반 통합 검색·멀티턴 대화, 태양광 발전량 예측 AI(TTA 95%), 채소류 영상 자동 측정 — 모두 우리가 직접 운영하는 현장에서 학습된 자체 모델입니다.
> GS인증 솔루션 10건, 특허 21건이 그 결과입니다.

### 03. AX로 환원 — Return as strategy.
> 우리가 만든 AI는 단일 모듈에 머무르지 않습니다.
> 제주특별자치도농업기술원의 농정 디지털 기반 전략, 드론 관측체계 3개년 ISP 컨설팅처럼, 우리가 검증한 기술은 곧 공공기관의 디지털 전환 로드맵이 됩니다.
