# About > History / 연혁

## 페이지 헤딩

**25년, 제주에서 시작해 도메인 운영·AI·AX로 진화한 여정.**

## 본문 (인트로)

2000년 제주 1세대 온라인 여행사 jeju114.com의 오픈으로 시작한 ㈜넥스트이지는, 25년간 단순 개발사가 아니라 도메인을 직접 운영하는 ICT 기업으로 성장해왔습니다. 2010년대에는 자체 솔루션(WEMS·MICE NICE·CMS 등)을 GS인증 등급으로 다수 출시했고, 2020년대 들어서는 AI·LLM 기반 농업·에너지 모델로 정부 R&D를 주관하며 공공 디지털 전환의 한 축을 담당하고 있습니다.

## 마일스톤 타임라인

### 2000~2010 — 창업·기반 구축
- **2000.10** 법인 설립 / **국내 최초 인터넷 여행사 jeju114.com 오픈** ← 회사 정체성의 출발
- **2002** 벤처기업 인증
- **2004** NEWSMAKER·이지샵 솔루션 개발 / 이노비즈(INNOBIZ) 1차 선정
- **2006** RFID e-Ticketing 시스템 선정 / 대한민국 e-비즈니스 대상 수상
- **2009** 정보통신공사업 등록
- **2010** 수출유망중소기업 선정

### 2011~2015 — 자체 솔루션·인증 본격화
- **2011** ISO 9001:2008 인증 / 가족친화기업 인증
- **2012** **eM-Maker · Sukso114 · WEMS** 등 GS인증 시작
- **2013** 소프트웨어공제조합 가입 / 산업디자인전문회사 등록 / **MICE-NICE v1.0** GS인증
- **2013.3** **베트남 자회사 NC9 설립** (호치민) — AMNOTE ERP/회계 SaaS 사업 개시
- **2014** Open API(개방형 티켓팅) 출원 / Ticket-Nice TiNi v1.0
- **2015** ISO 9001:2008 갱신

### 2016~2020 — 확장·R&D 성과
- **2017** 연구개발서비스업 신고 / ISO 14001:2015 인증 / 음식점 무선주차 호출용 AHP기반 추천 알고리즘 특허
- **2017** **4IR AWARDS 대상 수상** / **2018** 4IR AWARDS 2년 연속 수상(관광부문) / 제주경제상
- **2019** AMNOTE 회계솔루션 GS인증 / **베트남 모바일앱 10대 브랜드 선정** / **2019 4IR AWARDS 클라우드기획 부문 대상**
- **2020** 전자상거래 검증 특허 / 배달납 전자세금계산서 / **제주최초 골프통합플랫폼 섬프로(SUMPRO) 출시** (iOS/Android, 17개 골프장 제휴)

### 2021~현재 — AI·AX 본격 확장
- **2021** **제11회 제주 수출인의 날 수출유공기업 수출장려(IT 등 지식산업 분야) 수상** / 골프통합플랫폼 섬프로 상표권 등록
- **2022** MAINBIZ·벤처기업·가족친화기업 재인증
- **2023** 제주특회 글로벌투자유망 플랫폼 'SBS방방국락' 출연 / 누적 회원 4,000만 통과
- **2024** 김주경/강주경 대표이사 (사)제주ICT협회 11대 회장 선임 / **제주특별자치도농업기술원 — 제주 농업 디지털 영농지원 시스템 1단계 주관 (15개월·36억)**
- **2025.07** **제주 농업 디지털 영농지원 시스템 2단계 주관 시작 (16개월·18.9억)** — AI·LLM 기술 적용, DX·AX 컨설팅 동시 수행
- **2026** 누적 농업기술원 사업 31개월·54.9억 단일 사업 진행 중 / 강주경 대표이사 (사)제주ICT협회 활동 지속

## 강조 메시지 박스 후보

### Box A. "2000년에 만든 jeju114는 지금도 우리가 운영합니다"
> 제주 1세대 온라인 여행사로 출발한 jeju114.com은 25년이 지난 지금도 우리 회사의 사업부가 직접 운영하는 서비스입니다.
> 우리에게 관광은 사례가 아니라 현재 진행형의 사업입니다.

### Box B. "농업 디지털 전환 31개월, 단일 사업자로"
> 2024년 시작한 제주 농업 디지털 영농지원 시스템 사업은 1·2단계를 누적 31개월·54.9억 규모로 ㈜넥스트이지가 단독 주관하고 있습니다.
> 이는 단순 SI 수주가 아니라, 한 광역지자체의 농업 디지털 전환 전략을 민간이 함께 설계한다는 의미입니다.
